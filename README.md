This repository contains the files for [ufabric.org](https://ufabric.org) website.

## Projects feed

Public project progress reports are aggregated into `content/projects.md` from
`ufabric-org/projects-sync/sources.md` using:

```bash
make projects
```

Operations docs for this process:

- `ufabric-org/projects-sync/README.md`
- `ufabric-org/projects-sync/INTEGRATION.md`
- `ufabric-org/projects-sync/MAINTENANCE_RULES.md`
- `ufabric-org/projects-sync/NEW_PROJECT_SETUP.md`
